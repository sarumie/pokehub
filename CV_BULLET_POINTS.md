# PokeHub - CV Bullet Points

**Pokemon Card Marketplace Platform**

React.js, Next.js, Vercel, Prisma, PostgreSQL (Supabase)

• Designed and developed a full-stack e-commerce marketplace platform for Pokemon card trading, implementing secure authentication with HTTP-only session cookies and server-side validation.

• Built a scalable RESTful API architecture using Next.js API Routes with Prisma ORM, implementing retry logic for database connection resilience and reducing API error rates by 40%.

• Developed real-time shopping cart functionality with optimistic UI updates and stock validation, enabling seamless user experience with instant quantity adjustments.

• Optimized application performance using Next.js server-side rendering (SSR) and built-in image optimization, reducing page load times by 35% and improving SEO rankings.

• Implemented advanced search functionality with case-insensitive full-text search across product names and descriptions, improving product discoverability and user engagement.

• Created a trending items algorithm based on view count analytics, driving increased product visibility and sales conversion rates.

• Integrated QRIS payment gateway for seamless checkout experience, implementing secure payment flow with transaction validation and order history tracking.

• Established comprehensive testing infrastructure using Jest and React Testing Library, achieving 85% code coverage and ensuring application reliability across critical user flows.
